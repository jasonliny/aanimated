Assembly Code Animated

The purpose of this site is to familiarize users with assembly language programming through the use of animations to illustrate the functions of some MIPS commands.

This is not an accurate emulation of a MIPS system. Only a small subset of commands were implemented and they do not consider all the nuances of a MIPS system, but they can still be used to demonstrate some basic programs. This project is only meant to introduce users to how computers work by showing how different storage cells (registers) can accessed and manipulated to make programs.
